# Expense Tracker (Python)

## 📌 Project Overview

A simple Expense Tracker built using Python.

## Features

- Add Expense
- View Expenses
- Delete Expense
- Search by Category
- Highest Expense
- Lowest Expense
- Category-wise Total
- Save Data to File
- Load Data Automatically

## Technologies Used

- Python
- JSON
- File Handling

## How to Run

```bash
python expense_tracker.py
```
